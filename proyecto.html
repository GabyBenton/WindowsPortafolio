Pawsitive Vibes

-Descripción del Proyecto:

Pawsitive Vibes es una tienda en línea dedicada al bienestar de las mascotas, donde ofrecemos una amplia gama de productos seleccionados cuidadosamente para nuestros amigos peludos. Nuestro equipo de profesionales se compromete a brindar una experiencia de compra inolvidable a través de una plataforma que prioriza la felicidad y el bienestar animal.

-Diseño y Estilo:

Pawsitive Vibes se distingue por su diseño minimalista y acogedor, que resalta la esencia de la marca. Utilizando la herramienta de diseño Figma, creamos una interfaz visualmente atractiva y fácil de usar. Cada elemento visual se ha seleccionado meticulosamente para transmitir nuestra pasión por los animales y resaltar la gama de productos. El diseño elegante se combina con Bootstrap para lograr una experiencia responsiva y uniforme en todos los dispositivos, manteniendo el enfoque en los productos y la comodidad del usuario.

-Herramientas Utilizadas:

Nuestra plataforma Pawsitive Vibes se ha construido con una combinación de tecnologías robustas y líderes en la industria. HTML y CSS son la base de nuestro diseño, mientras que Bootstrap proporciona la estructura para una experiencia responsiva y estéticamente agradable. Utilizamos JavaScript para agregar interactividad y funcionalidad dinámica a la plataforma. La gestión de la base de datos se realiza a través de SQL, asegurando la seguridad y disponibilidad de la información crítica. Para una gestión eficiente de tareas y proyectos, adoptamos la metodología Scrum y utilizamos Jira como herramienta principal de organización.

Centro Comunitario

-Descripción del Proyecto:

En mi proyecto web, diseñé y desarrollé desde cero una página web completa para el Centro Comunitario CH29. Esta página web tiene como objetivo ofrecer una plataforma en línea que proporcione información detallada sobre los servicios y actividades que ofrece el centro para jóvenes interesados en su crecimiento personal, laboral y académico.

-Diseño y Estilo:

El diseño de la página web se caracteriza por su enfoque limpio y atractivo. Utilicé una combinación de colores sencillos pero vibrantes, como el blanco y el durazno, para transmitir una sensación de frescura y vitalidad. Además, incorporé animaciones de CSS para agregar interactividad y dinamismo a la experiencia del usuario.

-Herramientas Utilizadas:

Para llevar a cabo este proyecto, utilicé un enfoque combinado de HTML y CSS para estructurar y diseñar la página web. Utilicé HTML para crear la estructura de las diferentes secciones y páginas, y luego empleé CSS para dar estilo y formato a la página, incluyendo colores, fuentes y animaciones.
